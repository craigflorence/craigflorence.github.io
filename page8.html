Soporte Informático

La gestión informática adecuada puede ser difícil con mucho personal o poco tiempo. El servicio de soporte informático puede ayudar con la incorporación de nuevos empleados, las solicitudes de restablecimiento de contraseñas y la resolución de problemas generales del ordenador, entre otras tareas.

> Microsoft Server 2012 - 2019
> Apple MacOS
> Apple iOS, iPadOS, TVOs, WatchOS
> Sistema operativo móvil Android

Microsoft Azure y Microsoft 365

Azure es la oferta en la nube de Microsoft. Mis habilidades están con Azure Active Directory, gestión de grupos, gestión de grupos de recursos, Azure ADDS, máquinas virtuales, Azure Virtual Desktop, redes virtuales, NSGs y Exchange Online. También puedo gestionar instancias de servidor Microsoft físico (controlador de dominio, servidor Exchange).

Microsoft Office 365 y Microsoft 365 ofrecen las herramientas de prevención de pérdida de datos (DLP) y productividad más robustas y seguras del mundo. La integración de la suite Azure de Microsoft con el portal Office 365 significa que sus documentos estarán protegidos, pero también fácilmente compartibles para maximizar la productividad

Redes y Seguridad

Desde los interruptores y la gestión de cables en las instalaciones hasta las vLANs, el subneteo y el filtrado de puertos MAC, puedo ofrecer soporte remoto o en las instalaciones. Estoy familiarizado con los equipos Netgear, Ubiquiti, HP y tengo experiencia con los equipos Meraki, si quieres el hardware más premium.

Para las nuevas empresas puedo sugerir el mejor diseño de red, topología y adquisición de equipos.

También tengo experiencia avanzada con varias ofertas antivirus como SOPHOS, CiscoAMP y ESET.

Diseño de oficina, Startup (EDRA) y VoIP

Muchas empresas nuevas empiezan fuerte, pero luego se enfrentan a limitaciones por sus elecciones pasadas a medida que crecen. Puedo ayudarte a adoptar las mejores prácticas para asegurar que tus sistemas escalen contigo, si tengas 1 o 1000 empleados.

También puedo ayudarte con la migración de tus datos existentes desde DropBox, Google drive, Yahoo Mail, Gmail a todo el entorno de Microsoft

Mi familiaridad con las soluciones VoIP significa que incluso puedo configurarte un centro de llamadas o teléfonos de oficina simples con terminales físicos o teléfonos virtuales.

Diseño de sitios web

Puedo configurar, diseñar y gestionar un sitio web sencillo para tu empresa. Si necesitas dominios, certificados SSL, también puedo encargarme de ellos.
